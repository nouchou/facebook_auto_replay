Politique de confidentialité

L’application Facebook Auto Replay utilise uniquement les données fournies par l’API Meta (Facebook) afin d’automatiser les réponses aux messages Messenger et aux commentaires Facebook. Nous respectons la réglementation GDPR ainsi que les règles imposées par Meta Platforms.

Collecte des données

Nous collectons uniquement les informations suivantes :

  • ID de la page Facebook
  • Messages envoyés à la page via Messenger
  • Commentaires publics sous les publications de la page
  • ID de l’utilisateur Facebook interagissant avec la page

Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Générer des réponses automatiques via Messenger
  • Répondre automatiquement aux commentaires publics
  • Améliorer les performances du chatbot

Aucune donnée n’est partagée, vendue ou transmise à des tiers.
